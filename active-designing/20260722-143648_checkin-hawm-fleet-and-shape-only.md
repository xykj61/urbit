### What already is (checkable)

### Jail truth

An agent **inside** ai-jail can already: cross-build, pack APKs, run device-free witnesses, author Glow/Rye. It cannot honestly own USB attach or (usually) KVM device nodes the way the outer host can. Autonomy that *boots phones* stays **host-onpath** (or a future jail with explicit KVM/USB policy) — same discipline as HAWM0/TUBE install proofs.
