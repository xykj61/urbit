## ZETA6 — ret is newline; ↑ sends

**Today (GREEN):** `.@"return"` → Dexter `.submit`; bare ret paints JABS2 `chat ·`.

**Seated design:**

| Key | Action |
|-----|--------|
| `ret` | Insert newline (multiline Dexter buffer) |
| `↑` (new chrome key, bottom row beside space/ret family) | `.submit` → JABS2 chat / held line |
| Audience strip tap | Future: cycle faces (N-cycle; N=2 first) — not bare ret |

`LineEditor` grows honest multiline (or stores `\n` in the existing buffer with paint showing a soft break). Skate Dexter row may show the last line + `¶` count until a taller compose face exists — honest thin edge.

**Witness before narrative:** device-free — newline does not submit; ↑ submits; empty ↑ still paints `chat ·`.
