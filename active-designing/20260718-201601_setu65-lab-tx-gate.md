## Claim

When the SETU6 lab card is **stand-in**, Desk **refuses** framed TX. When the card is **armed** (`usb0` / `ncm*` / `enx*` CDC-NCM), Desk prepares the SETU5 framed payload and attempts an iface-bound UDP datagram on port **38503** to the iface default gateway (Pixel tether peer).

## Thin face (approved)

| Path | Behavior |
|------|----------|
| Stand-in / no iface | Gate refuses · host SKIP GREEN |
| Armed + iface | Prepare frame · `SO_BINDTODEVICE` · UDP unicast to gateway |
| Bind/send incomplete | Honest GREEN naming `send-failed` (CAP/gadget) |

**Live-sent GREEN** `20260720.233805` on Pixel 10a GrapheneOS USB tether (`enx*` · `cdc_ncm`) — brief [`20260720-233805_setu65-live-sent-enx-green.md`](20260720-233805_setu65-live-sent-enx-green.md).
