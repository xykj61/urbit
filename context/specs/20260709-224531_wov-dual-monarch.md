## 1. What This Spec Decides

Dual monarch means **two named Kumara pubkeys** must both attest the same root bytes before holders treat that root as current. It does **not** mean:

- three or more signers (Mycelium / M4),
- a new consensus protocol,
- a change to exit-bundle grammar or conservation,
- a second TigerBeetle cluster.

Solo monarch (lap 1 + seam A) stays valid forever for single-operator instances.

## 3. Two Envelope Shapes (Claude Weighs)

Exit honesty §3 already sketches both. Lap-open picks one:

| Shape | Form | Pros | Costs |
|-------|------|------|-------|
| **D1 — Twin log lines** | Two consecutive `wov:root` facts over **identical** canonical body bytes; each signed by one monarch | Minimal new grammar; verifiers already walk the log | Ordering / pairing rules; partial publish if one line lands alone |
| **D2 — Two-sig envelope** | One log line whose body carries `sig_a` + `sig_b` (or a small multi-sig wrapper) over one digest | Atomic publish; one root seq | New parse path; golden fixtures change |

**Bench leaning:** **D1** for the first dual witness — reuse lap-1 fact bytes; require both lines present before `verify_latest_root` advances. Graduate to D2 only if twin-line pairing proves awkward in practice.

---

## 4. Verification Rules (Unchanged Honesty)

When dual-control is live for an instance:

1. Instance config names exactly two monarch pubkeys (`monarch_a`, `monarch_b`) and the envelope shape.
2. A root at seq `N` is **accepted** only when both signatures verify over the same body (D1: two lines; D2: one envelope).
3. Exit-bundle SHA3 still equals `state-root-hex64`; holders prove balances exactly as lap 1.
4. `accept_exit` still burns against the last **accepted** dual root; a solo-signed root is refused once the instance is dual-configured.
5. If one monarch goes silent after a published dual root, holders who hold the bundle still exit — same availability promise as solo.

Migration: an instance may publish a one-time `wov:dual-enable` (or config resin) naming the second key; until then, solo rules apply. Exact memo shape waits lap-open.
