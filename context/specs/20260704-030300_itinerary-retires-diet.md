## Craft checklist for new prose

1. **Itinerary** names the route; **open itinerary** when every gate is cleared now.
2. **Stops** are exhaustive in prompts; **journey's pause** closes the list.
3. **Smallest-first** orders stops; **witness-first, red-then-green** proves each stop.
4. **Off-route** is the forbidden move — never *forage*, never *empty plate*.
5. **Lap** names an SLC-sized whole; stops wire witnesses that close laps.
