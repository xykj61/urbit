## For an Acme Corporation Employee, Reading Slowly

This is not a task list. It is a set of documents worth letting sit in the back of your mind — read once carefully, then carried into a walk, a shower, a hammock, whatever gives a hard question room to settle before it needs an answer. Every document below names a real fork rather than a foregone conclusion; none of them commits you to anything yet. Read them in this order, since each builds on what came before.

## The Reading Set, in Order

1. **[`context/specs/20260713-214400_urbit-parallel-naming-mapping-proposal.md`](../context/specs/20260713-214400_urbit-parallel-naming-mapping-proposal.md)** — start here. ## The Question Worth Carrying

Not "which of these should we build first" — that is a task-list question, for later, wide awake, at a keyboard. The hammock question is quieter: **does fusing this project's own hard-won safety discipline with Hoon and Nock's very different tradition make both traditions stronger, or does it quietly cost each of them the thing that made it worth borrowing from in the first place?** No document in this set answers that. That answer is yours to reach, in your own time, and it will be worth more for having been sat with rather than rushed.
