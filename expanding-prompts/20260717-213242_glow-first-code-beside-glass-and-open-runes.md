**Run this first** ‚Äî fold-shaped, already on the desk, already witnessed:

```bash

```

Source (`glow/gen/face-mix.glow`):


Why this one: `(mix ‚Ä¶)` is bitwise XOR of two faces ‚Äî the same *combine two values into a next root* shape the Neth/Sala fold story uses as a stand-in. It sits **beside** the glass season (session-root / fold vocabulary) without pretending to paint Skate or drain Dexter keys.
