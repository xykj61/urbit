## First Glow code you can write (and already have)

**Run this first** — fold-shaped, already on the desk, already witnessed:

```bash
rishi/bin/rishi run tools/glow_run.rish glow/gen/face-mix.glow
```

Source (`glow/gen/face-mix.glow`):

```
=/  next-root=@u32  (mix root x)
```

Why this one: `(mix …)` is bitwise XOR of two faces — the same *combine two values into a next root* shape the Neth/Sala fold story uses as a stand-in. It sits **beside** the glass season (session-root / fold vocabulary) without pretending to paint Skate or drain Dexter keys.

Smaller emitters exist (`bound-tick.glow`, `lent-tick.glow`) if you only need to prove the hop. Desk suite: `rishi/bin/rishi run tools/glow_run_desk_witness.rish`.

**Next named language rung:** **STOA15** (payload · multi-arm · library emit) — after **STOA14** thin `|%` GREEN `20260718.224030`. Living ladder: waymark **STOA** · compressor [`../docs/STOA.md`](../docs/STOA.md).
