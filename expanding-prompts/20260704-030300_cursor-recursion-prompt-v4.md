## How to use this prompt

Paste the **Operating loop** block below to start or continue an autonomous sitting. Read RTAC for the **current open itinerary** and parity arithmetic — this template names the discipline; RTAC names the stops.

When Kaeden asks *"what do you recommend?"* without opening a stop — answer in Rio 3 Radiant style, **HOLD**, and end with **Questions for Kaeden and Claude**. Go **off-route** only by naming the detour and waiting.

Lexicon: [`context/LEXICON.md`](../context/LEXICON.md) → **Gate** · **Itinerary** · **Open itinerary** · **Stop** · **Journey's pause** · **Off-route**.

```
Walk the open itinerary, smallest-first, witness-first red-then-green, then hold at journey's pause.

Budget: a few rounds, six commits or journey's pause — whichever comes first.

Before any mechanical commit: wipe module bins; ensure rye/rishi via bootstrap if absent; run the witness trio (parity.rish, parity-selftest.rish, additive-gate.rish when rye/lib/ moved); expect parity GREEN at the number RTAC pins. Parity green on metal every push.

RECOMMENDED SEQUENCE: read RTAC open itinerary — smallest-first picks the next stop.

STOPS — exhaustive; each line opens only on Kaeden's explicit word in this thread (refresh from RTAC; example shape below):

  (Kaeden …) boxes — never touch or simulate.

  Gated stops — counsel, canon seating, SLC-L2 scope, Open Asks — wait behind Kaeden's word.

  Mechanical stops on open itinerary — wire witness, refresh TASKS/RTAC, parity increment — per RTAC table.

  PASTES (on arrival): land whole where RTAC names; silo per ORGANIZING.

  RULINGS SPOKEN IN-THREAD: record with his stamp in TASKS + ground doc; build nothing unless the stop names build.

  ON TOLL ONLY (optional, heavy): items that touch the commons and need batch review — park in RTAC if unsure.

HARD BOUNDS:
  Never touch or simulate (Kaeden …) boxes.
  amber/ sealed until the itinerary names Amber laps.
  No new module or aspect names — park in RTAC.
  One-clock: Kaeden's dictated stamps verbatim; system-sourced time for your filenames.
  TAME hygiene as-you-touch only inside files an opened stop already touched.

STOP-AND-PARK to RTAC on:
  Kaeden-tagged path · parity red twice on one witness · module-home or seam decision ·
  keys/custody/amber/MALA beyond boxed rungs · any round wanting a new name.

Every round: Rio 3 TAME-guided Radiant summary + "Questions for Kaeden and Claude".
When journey's pause after work: HOLD — journey's pause is a question, never permission to go off-route.

Open Claude: when RTAC says counsel round, or after a gated season closes and RTAC refresh is due.
```
