## The Repo as a Professional Reference

Kaeden's aim is a repository that teaches: every bench documented, every lap recorded beside the roads considered around it, and guides and templates ready for anyone who clones the tree to build their own Simple, Lovable, Complete things upon it. The tree already holds the spine of this. The manual program stands shaped as four rooms with tutorials first — because tutorials can only teach what already runs — and numbered, foundation-grade reference specs graduating one by one, beginning with the Aurora, Tally, and Caravan interfaces. The outside world confirms the shape from two directions. The Diátaxis framework, now the common tongue of professional documentation, divides all docs into the same four rooms — tutorials, how-to guides, reference, and explanation — each serving a different reader's moment; our manual arrived at the division on its own feet. And the architecture-decision-record tradition holds that every significant decision deserves a small standing document naming its context, **the options considered**, the choice, and its consequences — which is precisely the "alternatives beside laps" Kaeden asks for, already half-practiced in every session log that names a road taken and a road set aside.

Four counsels follow, each small. First, let every lap record carry a named *roads considered* passage — one paragraph naming what was weighed and why it rested — so a cloner inherits the reasoning, never only the result; the decision-record discipline in Radiant voice. Second, promote the bench logs into a professional reference of practice: the reverse-chronological README tables already index them, and one living page gathering every bench by name, dates, and itinerary would complete the shelf — the benches *are* the documentation of how this tree is actually grown. Third, let the raise recipe graduate: the moment the current recipe is seated and reproduced by a stranger's hands, it becomes tutorial one — an executable document, tested the way the wider world tests reproducible builds, where the doc that raises the bench *is* the proof the doc is true. Fourth, ship three templates, each Simple, Lovable, Complete at its own size: a template lap (witness first, red then green), a template session log, and a template proposal packet — so a cloner's first contribution has a mold waiting.

The framing beneath all four: cloners are users, and documentation is the interface they hold. A repository whose guides are tested, whose decisions carry their alternatives, and whose history reads aloud well is a product — and the release-notes culture of the toolchain we pin shows how far that can go, where each version's notes are themselves a teaching document strangers cite for years.

## The Practice Ground

Kaeden asks whether Scribble should expand into a testnet development platform — testMALA, or a better name — where practice value flows freely before real value does. Two honest observations shape the counsel. Scribble stands named for the inference season, its shape deliberately open, beside Lantern, Lattice, and Anvil; loading it also with testnet duty would give one module two ideas before it has fully received its first, and one module, one idea is the oldest rule in the root. And in the ledger world, a testnet is rarely a separate product at all: the practiced pattern is *the same software under a different genesis* — a local network, a faucet that pours free practice tokens, and a policy wall keeping play value and real value from ever touching. Read that way, the practice ground is mostly an **aspect of MALA itself** — a rehearsal genesis, a faucet verb, and Pond's policy enclosure — rather than a new organ; the platform is the configuration, and the configuration is a `.brix` descriptor away.

If a named practice ground is nonetheless wanted — and a warm name would serve the creators and cloners it welcomes — these park in RTAC on Kaeden's word:

| Candidate | Its field | Honest note |
|---|---|---|
| **Lila** | Sanskrit: divine play | Play money in the truest, most devotional sense; short and luminous |
| **Greenhouse** | Where seedlings grow before the field | Agrarian; sits beautifully beside Linengrow and Granary |

| **Chalk** | Marks that practice and wipe clean | The schoolroom slate's partner; modest and clear |

| **Rehearsal** | Plain and honest | Says exactly what it is; least poetic of the set |

The seat, as always, is Kaeden's alone; testMALA remains serviceable until a name is spoken.
