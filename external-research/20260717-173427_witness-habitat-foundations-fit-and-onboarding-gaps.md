## A Better Name for Happy-Zone Testing

The foundation keeps J.B. Rainsberger’s phrase **happy zone** (gratitude: `gratitude/IntegratedTestsAreAScam.md`). That warmth belongs in foundations and history. For Acme Corporation documentation — serious, reference-grade — the tree already supplies a clearer professional synonym:

| Name | Meaning | Use |
|---|---|---|
| **Witness habitat** (recommended Acme face) | The fast, isolated place where pure folds and seam contracts are proven | Tutorials, expanding-prompts, TAME when pairing seats |
| **Fold core** | Same idea, stack-native | Technical internals |
| **Seam-proven center** | Emphasizes mirrored collaboration/contract pairs | Method chapters |
| **Happy zone** | Historical / foundations / gratitude | Never rewrite dated seats; cite beside the Acme face |
| **Thin edge** + **metalsmoke** | Already professional | Keep for the few honest host/device checks |

**Recommendation:** seat **witness habitat** as the Acme-facing synonym in LEXICON and new prose; keep **happy zone** as the foundation’s own name and teacher attribution. Do not rewrite `20260702-165412`.

### One-line Acme definition

**Witness habitat** — the fast, isolated suite of checks that prove pure folds and seam contracts; the outside world (devices, networks, hosts) stays on a thin edge with a few metalsmoke proofs.

## Is Documentation Complete Enough for Cursor Development?

**Operational get-started is real. A foundations-and-rules tutorial path is not yet whole.**

### What an employee can already find

- Manual four rooms (`manual/README.md`) · Glow-OS get-started (language + device)
- First-witness tutorials · Rishi reference
- Always-on Cursor rules (Quin · Radiant · TAME · session-logs · align · send · collaboration · Acme voice)
- Compass · grain · LEXICON loop words (gate · itinerary · lap · nib)

### What is still missing (honest gaps)

| Gap | Why it matters |

| Sequenced **Understanding → first Cursor day** tutorial | Foundations + grain + witness habitat + compass + session-logs are scattershot, not a path |
| Explicit **Cursor rules chapter** for humans | Rules load for agents; employees need an ordered reading |

| Full **Glow language** tutorial | Correctly blocked on a general parser |
| Happy-zone **pairing in TAME** body | Still horizon |

**Recommendation:** open `manual/tutorials/cursor-and-the-compass.md` as the stitched path (scaffold → checkable as each section earns a witness). Do not invent a Glow beginner textbook ahead of the parser.

## Cycles, Loops, and Logging — the Vocabulary Already Seated

| Word | Job |
|---|---|
| **Gate** | What must be true before a lap opens |
| **Itinerary** | Named sequence of laps |
| **Lap** | One bounded, witnessable unit of work |
