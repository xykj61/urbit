## Q1 — Caravan Hardening, Ranked Rather Than Left Open

Last sitting named four candidates and ranked none; asked to choose, this bench chooses.

**First: the `witness_stop_after` footgun.** Smallest possible change, pure safety net, and it is the only one of the four that protects against a mistake rather than merely proving a gap — a test-only argument reaching a real invocation by accident is exactly the kind of quiet failure this tree's own discipline exists to prevent. Free insurance, first.

**Second: source-crash recovery.** This is the real gap among the four. The supervisor's own witnesses have proven what happens when the *fetcher* falls; nothing yet proves what happens when the long-lived `source-loop` itself dies mid-service — and a silent hang is a worse failure than a loud one, which is precisely the shape of thing worth closing before it is discovered the hard way.

**Third: host-mirror under supervision.** Each half is proven alone; nothing proves them together, and a crash mid-list is a real, if smaller, question than the first two.

**Held back, not abandoned: extended-run stability.** Every current witness is a small bounded run by design, and nothing in this tree is meant to run unattended and indefinitely yet — the outer scheduling ring stays its own later word, exactly as the graceful-stop counsel left it. Testing "dozens of cycles" today would prove the same shape of claim a short run already proves, only longer, without a genuine indefinite consumer on the other end to make the extra length mean something. Worth doing the day one exists; not worth doing to fill a slot today.
