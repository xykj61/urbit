# Wayland — libwayland-client

**License:** MIT (Expat), quoted below  
**Role for us:** Host-seam display library for the **hosted** Brushstroke build — links on the development machine (Ubuntu GNOME Wayland / Mutter today). A dependency of the hosted build, **never** of Rye OS proper on bare metal.

**Clean room:** We study and run Wayland; we never copy its source into our modules.
