## Digital Audio Workstation

Using REAPER with your PC, Mac, or Linux computer, and no other software, you can import any audio and MIDI, synthesize, sample, compose, arrange, edit, mix, and master songs or any other audio projects.

If you add a hardware audio interface and a microphone, you have a complete professional quality recording studio.

There is only one version of REAPER, used by home hobbyists and high-end professional studios alike. Evaluate the full version for up to 60 days.

## Record

- Record and overdub audio and MIDI; inputs limited only by hardware
- Dozens of supported audio formats, any sample rate or bit depth
- Any track can record input or processed output
- Multiple takes, overdub, punch in/out, loop record
- Arm/disarm without stopping playback
- Stereo, surround, or multichannel files, before or after FX
- Monitor with or without software FX; multiple monitor mixes
- Tape-style varispeed; simultaneous multi-disk recording

## Produce

- Drag and drop import, arrange, render
- Mix audio, MIDI, video, still image on any track
- Move, split, glue, resize, trim, loop, time stretch, pitch shift, fade, crossfade, slip, snap — without switching tools
- Nested folder system for group editing, routing, bussing
- Multiple projects in tabs
- Full automation for tracks, items, plug-ins
- Tempo, time signature, varispeed; takes and lanes for comping
- Up to 128 routable channels and 128 MIDI buses per track

## Transform

- Plug-in formats: VST, VST3, LV2, CLAP, DX (Windows), AU (macOS), JSFX
- Real-time or rendered FX; JSFX scripting engine with hundreds of effects
- ReWire; automatic PDC; multiprocessor utilization
- Plug-in bridging and crash firewalling; sidechain any plug-in
- Pitch/time engines including Elastique 3 Pro; surround panners
- Virtual outboard, MIDI utilities, NINJAM, streaming plug-ins

## Render

- Any combination of tracks, regions, items; batch thousands of files
- Queued render instructions; dither, LUFS normalization, true peak limiting
- HTML render statistics; Region Render Matrix; wildcard file names
- Metadata formats; multi-core batch conversion with CLI

## Customize

- Colors, icons, toolbars, layouts, screensets
- Dockable windows; thousands of actions bindable to keys, MIDI, OSC
- Extensible via Lua, EEL, Python (ReaScript); integrated script IDE

## Intangibles

- Fast start; edit while playing/recording
- Portable USB install (~20 MB installer)
- Active forum; community user guide by Geoffrey Francis

## Technical highlights

**OS:** Windows, macOS, Linux (GTK+3; x86_64, i686, armv7l, aarch64)

**Audio APIs:** ASIO, WASAPI, Core Audio, ALSA, JACK, PulseAudio; up to 512 remappable I/O channels

**Formats:** Extensive read/write including WAV, FLAC, MP3, OGG, MIDI, video containers

**ReaPlugs included:** ReaEQ, ReaComp, ReaXComp, ReaDelay, ReaLimit, ReaPitch, ReaTune, ReaFIR, ReaGate, ReaVerb, ReaSynth, ReaSamplOMatic5000, and others

**JSFX:** User-scriptable 64-bit audio/MIDI; edit on the fly without stopping playback

**Hosting:** VST2/3 (ARA), LV2, CLAP, DX, AU, JSFX; oversampling; auto-bypass on silence
