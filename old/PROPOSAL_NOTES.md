# `old/` — This Proposal's Predecessor Material

**Language:** EN
**Style:** Radiant (see `../context/RADIANT_STYLE.md`)

---

This folder holds everything that occupied `~/urbit` before this proposal's own content took the top level — kept whole, never discarded, per this project's own "accrete, never break" root discipline. Glow OS is a **downstream research repository developed as a collaborative proposal to Urbit**, so it descends directly from `urbit/urbit` and keeps its ancestor's tree in full view rather than erasing it. Each subfolder keeps its own original `README.md` untouched; this file is this project's own addition, naming what sits here and why.

- **`urbit/`** — the original `urbit/urbit` tree as cloned fresh at this repository's founding commit. This project's own `.git` history at the repository root *is* this tree's history; the working files here are the snapshot at the moment the downstream work began.
- **`nockchain/`** — a pre-existing personal clone of `zorp-corp/nockchain`, unrelated to this proposal, present on this machine since October 2025 and nested here rather than deleted.
- **`urbit.org/`** — a pre-existing personal clone of the `urbit.org` website source (the same content behind `docs.urbit.org` and the public Urbit docs), also present since October 2025. Its `content/overview/` and `content-old/getting-started/` folders are the most relevant reading for anyone adapting Urbit's own onboarding structure to this project's — see [`manual/guides/macos-ai-jail-setup.md`](../manual/guides/macos-ai-jail-setup.md) for this project's own onboarding equivalent.

None of these three histories were merged into this project's own `.git` — each keeps (or, for `urbit/`, shares) its original history, recoverable in full at `~/backups/urbit-mirror.git`, `~/backups/nockchain-legacy.git`, and `~/backups/urbit.org-legacy.git` respectively.
