## Decisions

- **Sentinel-terminated paths via `allocPrint`**: Rye's Dir methods require `[:0]const u8` for paths. Rishi strings are `[]const u8`. Using `std.fmt.allocPrint(allocator, "{s}", .{path})` produces a `[:0]u8` in the garden, freed at exit.
- **`write-file` as a statement**: It is an effect (writes to disk), not a computation that produces a value. The `say` and `assert` precedent holds: effects are statements.
- **`list-dir` returns all entry names**: Files, directories, symlinks — all returned. The user can filter with `where` if needed. Bounded at 256 entries.
- **No `delete-file` builtin yet**: Not in scope. Test files written to `/tmp` are ephemeral.
