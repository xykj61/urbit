## Decisions

- **Caller-provided backing memory** — the region holds no ownership, just a view. This is the right model for a garden allocator: the caller decides where memory comes from; the region decides where within it things land.
- **`clear` does not zero the memory** — the region guarantees bounds, not erasure. A future hardening pass could add an optional zero; for the seed, the invariant is enough.
- **Stack backing in `main`** — a `[64]u8` on the stack is simple, auditable, and avoids the arena for a self-contained demonstration. Real use will pass in a region of an arena or a mmap'd buffer.
- **Hosted first** — freestanding Tally on RISC-V follows Aurora's relay pattern: host proof first, bare metal after.
