3. Wrote `external-research/971_editors_remoting_tame.md`:
   - **The editor question**: Kakoune's select-then-act is the most TAME-aligned model. No new DSL — Brix for settings, Rishi for behavior, Rye for the implementation.
   - **Three languages, not four**: Brix (declare), Rishi (behave), Rye (build). No VimL, no Lua, no KakouneScript.
   - **SSH vs Mosh vs Comlink**: SSH is encrypted by default (good) yet TCP-blocked and byte-streamed (not TAME). Mosh is state-synchronized over UDP (TAME-aligned) yet bootstraps through SSH. Comlink remoting uses our own crypto, value-based state sync, Pond enclosures, and bounded messages.
   - **Editor plugins run inside Pond enclosures**: bounded, isolated, policy-as-a-value. A misbehaving plugin cannot corrupt the editor.
